• CSS (Cascading Style Sheets)
  • CSS는 HTML 요소의 스타일을 정의하는 스타일시트 언어입니다.
  • 웹 페이지의 레이아웃, 색상, 폰트, 간격 등을 설정하여 시각적으로 멋진 페이지를 만듭니다.
다음은 CSS의 주요 역할입니다
  1. 스타일 적용: HTML 요소의 시각적 스타일을 정의합니다..
  2. 레이아웃 설정: 페이지의 전체 레이아웃과 개별 요소의 위치를 설정합니다..
  3. 반응형 디자인: 다양한 디바이스에서 적절히 표시될 수 있도록 반응형 디자인을 구현합니다..
